# 航空性能计算器

隐身飞翼气动性能估算工具。内置 7 款机体与 7 款发动机数据库，支持 CSV 导入/导出自定义数据。

::: tip 使用提示
- 计算器内置了默认的机体和发动机数据库，可直接使用
- 点击右侧 **Database Management** 面板的 **Export CSV** 可下载当前数据库
- 点击 **Import CSV** 可上传自定义数据库（将与现有数据合并）
- 左侧 **Load Airframe Template** 下拉菜单可快速加载预设机体参数
:::
