# 追逐太阳：一块光伏瓦的三万六千公里奥德赛

**作者：** 艾伦·麦克马洪 (Alan McMahon) **发自：** 得克萨斯州，博卡奇卡 **首发于《时代》周刊 (TIME) 2041年11月特刊：改变地球的100个瞬间**

在成为全人类仰望的“新神”之前，它仅仅是一个编号——**Module #14092**。

它是一个直径8米的完美六边形，厚度却不及一本精装版的《大英百科全书》。此刻，它正悬浮在距离赤道三万六千公里的地球静止轨道（GEO）上，静静地沐浴着未经过任何大气层过滤的、暴烈而纯粹的恒星光芒。

它是组成“天照号”（Amaterasu）这座直径五公里的巨大人造太阳的四十万个细胞之一。但如果追溯 Module #14092 的身世，你会发现，它的旅程不仅是一场对抗引力与真空的史诗，更是美、日、英三国在21世纪中叶进行的一场极其冷酷、精密且史无前例的工业合谋。

### 第一幕：在名古屋与巴罗的锻造

Module #14092 的骨肉诞生于太平洋的西岸。在日本名古屋港的三菱重工飞岛工厂——一个曾经制造H3火箭的无尘圣殿里，特种碳纤维被编织成极其轻盈却坚韧的桁架结构。信越化学提供的超高转化率柔性薄膜太阳能电池，像给初生婴儿穿衣般，被极其平整地贴合在骨架的一面。

“它轻得像一片羽毛，但在真空中，它能承受住近乎永恒的拉力。”三菱的首席材料工程师在接受《时代》采访时如此评价。

与此同时，它的大脑和神经，却在地球另一端的阴雨中成型。在英国巴罗因弗内斯（Barrow-in-Furness），BAE系统公司那座原本用来建造核潜艇的巨大船坞大厅里，军工级的相控阵微波发射器正在进行组装。数以百万计的射频芯片被封装在具有极致抗辐射能力的基板中，它们决定了太空中收集到的电能，能否以绝对精准的高斯波束，跨越三万六千公里的虚空，毫厘不差地投射到洛杉矶的接收网格上。

随后，这两半截然不同的工业结晶，被装上了特制的远洋滚装船，跨越巴拿马运河与大西洋，向着同一个咽喉之地汇聚——美国得克萨斯州，布朗斯维尔港。

### 第二幕：德州的“巴别塔”与螺旋弹夹

在布朗斯维尔的联合总装超级工厂，来自日本的“光伏之骨”与来自英国的“微波之脑”完成了最终的结合。但这并非旅途的最难点。真正的挑战在于：如何将这脆弱的巨物送入太空？

SpaceX 给出的答案是粗暴的“弹夹式”发射。

在巨大的厂房内，Module #14092 被起重机吊起，与另外 4999 个一模一样的兄弟姐妹一起，被塞进了一个巨大的圆柱体发射夹（Cartridge）中。

为了抵御星舟（Starship）在穿越最大动压点（Max-Q）时那足以撕碎钢铁的低频声学震动，英美日三方的工程师达成了一项令人惊叹的拓扑学妥协：这 5000 个模块并没有完美对齐。Module #14092 相对于它底下的模块，被极其精确地旋转了 0.072 度。

5000 个模块摞在一起，形成了一个极缓的螺旋。随后，六根贯穿整个弹夹的镍钛记忆合金拉杆从边缘收紧，施加了两百吨的预紧力。这一刻，5000 片脆弱的“薯片”在物理上化作了一个不可摧毁的刚性金属圆柱。

几天后，在几公里外的博卡奇卡星际基地（Starbase），巨型机械臂“麦加吉拉”将这个装满人类能源野心的“弹夹”塞入了星舟 V4 的腹部。伴随着猛禽发动机震碎云层的轰鸣，Module #14092 告别了地球。

### 第三幕：真空中的奥德赛与最后一次洗牌

仅仅八分钟后，疯狂的震动平息了。星舟的货舱门在近地轨道（LEO）的死寂中缓缓打开。

接手的，是 Maxar 公司的“克莱兹代尔马”级（Clydesdale）重型轨道拖船。这艘没有乘员的机械巨兽，伸出机械臂，稳稳地抓住了整个弹夹。伴随着尾部喷吐出的幽蓝色液氩等离子体，拖船开始了长达三个月的缓慢爬升。

这是一场没有声音的奥德赛。没有轰鸣，只有牛顿力学在真空中的绝对统治。

最终，在距离地球三万六千公里的赤道上空，拖船抵达了“天照号”的施工阵位。此时的空间站，中心缆索已经就位，但庞大的阵列才刚刚显露雏形。

拖船将整个弹夹精准地对接在正在边缘游走的“连续移动组装机器人”的馈料口上。

接下来的画面，堪称现代工程史上的神迹。

系统向弹夹边缘的六根记忆合金拉杆通入电流。伴随着热相变，两百吨的预紧力瞬间冰消瓦解。组装机器人的机械爪极其轻柔地探入弹夹，就像赌场里最老练的荷官发出一张扑克牌一样，将 Module #14092 抽了出来。

机械臂翻转，平移，将其推入碳纤维骨架的边缘。

_“咔哒。”_

虽然真空中没有声音，但在地球调度中心的遥测数据中，这声机械锁扣的咬合无比清脆。通信总线接通，冷却回路连合。Module #14092 上的薄膜光伏瞬间被激活，它开始向中心电网输送第一缕纯净的直流电。

它成为了“神”的一部分。

### 第四幕：光速的套利与和平的武器

Module #14092 及其兄弟们的旅程之所以能够成行，绝不仅是因为工程师的浪漫，更是因为资本的嗜血与人类对生存的渴望。

“天照号”是一场豪赌。它的初期研发与总资本支出（CAPEX）高达令人咋舌的 430 亿美元。但在《时代》获取的内部财报模型中，这台机器本质上是一台悬浮在太空中的“印钞机”。

它的商业逻辑被称为“跨时区日不落（Follow the Night）”。利用相控阵微波毫秒级切换波束的能力，在洛杉矶遭遇极端热浪、空调负荷拉爆电网的白昼，“天照号”将以高达每千瓦时 20 美分的“救急天价”向美国输送吉瓦级的能量；而当加州入夜，波束瞬间掉头，砸向正值工业用电早高峰的东京湾。

在这种极致的双城调峰套利下，430 亿美元的债务预计将在短短七年内被抹平。而当后续的“企业号”、“羲和号”相继落成，规模效应将把平准化度电成本（LCOE）暴力击穿至 3.3 美分/千瓦时。那将是化石能源时代的最终墓志铭。

今天，当你在洛杉矶的夜晚打开一盏灯，点亮它的能量可能在两秒钟前，刚刚打在 Module #14092 冰冷的硅片上。

大国们曾经为了地底的黑色液体流血漂橹。而现在，他们将最致命的军事雷达技术、最庞大的火箭与最贪婪的金融算法结合在一起，在太空中编织出了一张谁也离不开的能量之网。

这或许是“普罗米修斯-以太计划”带给人类最大的意外之喜：当所有超级大国的心脏，都依靠太空中同一条脆弱的光之脊索跳动时，和平，竟然成了唯一的物理学选项。
